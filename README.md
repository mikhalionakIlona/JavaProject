# 📌 Тема: Разработка REST API для управления фотосессиями
# 🎯 Цель работы 
 Разработать базовое REST API приложение для управления задачами, проектами и пользователями с использованием Spring Boot
📋 Задачи лабораторной работы 1
* Создать структуру многослойного приложения (Controller-Service-Repository)
* Реализовать модели данных: User, Project, Task, Tag, Comment
* Создать in-memory репозитории для хранения данных
* Разработать DTO для разных уровней доступа
* Реализовать мапперы для преобразования между сущностями и DTO
* Создать REST контроллеры с обработкой различных HTTP методов
* Добавить тестовые данные при запуске приложения
* Реализовать фильтрацию задач по статусу
